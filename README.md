# McStatus-API
这是一个 Flask API，主要封装了 mcstatus 包，用于查询 Minecraft 服务器状态，支持Java和基岩，以及附带其他小特性！

## 📦安装/💻使用

以下提到的方法 任选**其一** 即可

<details open>
<summary>uv</summary>

```bash
uv run app.py
```
  
</details>

<details open>
<summary>pdm</summary>

```bash
pdm install
pdm run app.py
```
</details>

<details open>
<summary>pip</summary>

```bash
pip install -r requirements.txt
python app.py
```
</details>
